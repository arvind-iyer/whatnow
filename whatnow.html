Reverse Engineer

Hunter S. Thompson, famous for his book "Loathing in Las Vegas" and inventing Gonzo journalism, typed out The Great Gatsby by F. Scott Fitzgerald and A Farewell to Arms by Ernest Hemingway on a typewriter to learn about writing styles of the authors and to feel what writing a novel was like.

In web development, you can apply similar learning method by looking into others' website codes using built-in browser functions such as 'Develop' > Show Web Inspector / Page Source on Safari, or Firebug add-on on Firefox.

In fact this website was built from scratch...mimicking this website using built-in Safari web development tools.

Programming

HTML/CSS will only get you as far as this website, which lacks user interactivity.

Learn Javascript to add interactivity.

Learn Ruby, Java and/or Python to learn server-side programming (back-end).

Learn C to learn fundamentals of software engineering and origins of higher level languages (Java, Python).

Although it is not necessary to know Java, Python or C to become front end web developer, learning software engineering and other programming languages will help you understand fundamental principles of programming.

Database
